ენის გადართვა

ენის გადართვა შესაძლებელია პროგრამის მენუდა ასევე კლავიშების კომბინაციით. ენის გადართვა სხვადასხვა მოდელებზე სხვადასხვანაირად არის განხორციელებული.
  • E61 / E63 / E70(qwerty) / E90(qwerty) ზე გადართვა ხდება ctrl + რომელიმე კლავიშის დაწოლით.
    ეს კლავიში ავტომატურად არის E61 / E63 ზე 'q' და E70 / E90 ზე 'p', თუმცაღა შესაძლებელია სხვა ასოს არჩევა.
    ეს ხდება საკონფიგურაციო ფაილიდან (C:\\Data\geoinput\config.cfg).
    დამატებითი ინფო პროექტის საიტზე.
  • N97(qwerty) / E71 / E72 / E75(qwerty) / 5730(qwerty) ზე გადართვა ხდება Space ის ზედიზედ დაწოლით.
  • სხვა დანარჩენ მოდელებზე და ზემოთ ჩამოთვლილი მოდელების ალფანციფრულ კლავიატურაზე
    გადართვა ხდება # და 0 ის თანმიმდევრობით დაწოლით.

მენიუ

  • გამონაკლისები

    • geoinput ზოგი პროგრამის მუშაობს ხელს უშლის.
    ზოგიერთ პროგრამაში საჭიროა ციფრების პრიორიზირება რაც geoinputს არ ესმის.
    მაგალითად კალკულატორში ან მაღვიძარას დაყენებისას.
    ასეთი პროგრამებე შესაძლებელია დაემატოს "გამონაკლისებში".
    როცა რომელიმე "გამონაკლის" პროგრამას ფოკუსი აქვს, geoinputი ატარებს კლავიშებს და არ ცვლის მათ ქართული ასოებით.

    • დამატება - "გამონაკლისის" დასამატებლად ვირჩვევთ გამონაკლისების მენიუდან "დამატება".
      დასამატებლად საჭიროა სასურველი პროგრამის არჩევა. სავარაუდოდ Telephone, Standby mode, Clock, Calculator და ა.შ.
    • წაშლა - გამონაკლისის წასაშლელად ვირჩევთ გამონაკლისების მენიუდან "წაშლა". სიიდან ვირჩევთ სასურველ პროგრამებს და ვაწვებიტ "OK".

  • კონფიგურაცია

    • სხვადასხვა პარამეტრები

    • ავტო ქარ. - პროგრამის გაშვებისას ავტომატურად გააქტიურდეს თუ არა ქართულის წერა
    • დამალვა - geoinput გაეშვას თუ არა როგორც სისტემატური პროგრამა. ამ შემთხვევაში geoinput არ გამჩნდება აქტიური პროგრამების სიაში.
    • ინდიკატორი - გადართვისას გამოჩნდეს თუ არა გააქტიურებული ენა (ინფო pop-up)
    • მარტივი მენუს მარტივი ვარიანტი. სრული ვერსიის დასაბრუნებლად საჭიროა შეიცვალოს პარამეტრი საკონფიგურაციო ფაილში(C:\\Data\geoinput\config.cfg)
